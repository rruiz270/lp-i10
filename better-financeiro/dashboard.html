⚠ NFs BMA não lançadas no Omie Better
A BMA emite notas mensais de ~R$ 200-350k para a Better (BPO folha/benefícios/impostos), mas essas NFs não estão registradas como Contas a Pagar no Omie Better. A coluna BMA mostra o gasto real via API BMA. Quando o financeiro lançar as NFs no Omie Better, a coluna BMA passará a ser apenas consulta.
